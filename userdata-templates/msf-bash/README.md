# 场景使用

1. 使用前请按照注意事项进行配置
2. 使用方法：安装完成后运行以下命令

启动 msfconsole：
```
msfconsole
```

更新 Metasploit：
```
msfupdate
```

# MSF 简介

Metasploit (MSF) 是全球最流行的渗透测试框架，包含：

- 漏洞利用模块 (Exploits)
- 辅助模块 (Auxiliary)
- 后渗透模块 (Post)
- 载荷生成器 (Payloads)
- 编码器 (Encoders)
- 等等

# 注意事项

- 本工具仅供 **授权的渗透测试** 使用
- 首次运行会初始化数据库
- 可使用 `msfdb init` 初始化数据库
