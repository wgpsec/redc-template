# Usage

1. Configure according to the notes before use
2. Usage: After installation, run the following commands

Start msfconsole:
```
msfconsole
```

Update Metasploit:
```
msfupdate
```

# About MSF

Metasploit (MSF) is the world's most popular penetration testing framework, containing:

- Exploit modules
- Auxiliary modules
- Post modules
- Payloads
- Encoders
- And more

# Notes

- This tool is for **authorized penetration testing only**
- Database will be initialized on first run
- Use `msfdb init` to initialize the database
