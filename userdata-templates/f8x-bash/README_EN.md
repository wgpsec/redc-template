# Usage

1. Configure according to the notes before use
2. Usage: Run the following command in terminal

```
./f8x
```

View help information:

```
./f8x -h
```

# About f8x

f8x is a penetration testing toolkit that can deploy multiple penetration testing tools with one click, including:

- Scanners
- Exploitation tools
- Backdoor generation tools
- Privilege escalation tools
- And more

# Notes

- This tool is for **authorized penetration testing only**
- Dependencies will be installed automatically on first run
